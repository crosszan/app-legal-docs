# Virtue 隐私政策

[**View in English**](virtue-privacy-en.html)

1. **简介**
   Virtue（以下简称"本应用"）由 **CrossZan** 开发运营。本政策说明我们如何处理您的数据。简而言之：我们不收集任何数据，所有内容均保存在您的设备上。

2. **我们收集的数据**
   - 本应用**不会**向我们或任何第三方传输、上传或同步任何数据。
   - 您在应用中输入的所有内容——包括习惯打卡记录、美德自检记录、每日反思以及设置——均通过 Apple 标准本地存储（UserDefaults）仅保存在您的设备上。

3. **数据使用方式**
   - 您的数据仅用于在您的设备上呈现应用功能。
   - 我们绝不将其用于广告、数据分析、用户画像或任何其他目的。

4. **数据存储与安全**
   - **仅限本地**：所有应用数据均保存在您的设备上，我们无法远程访问。
   - **无需注册**：本应用无需创建账号或登录即可完整使用。
   - **访问控制**：建议您启用设备密码、Face ID 或 Touch ID，防止他人未经授权查看您的记录。

5. **第三方数据共享**
   - 本应用不与任何第三方共享您的数据。
   - 本应用不包含任何第三方 SDK、广告库或数据分析框架。

6. **儿童隐私**
   - 本应用不会有意收集 13 岁以下儿童的数据。
   - 如果您认为儿童在未经许可的情况下录入了数据，您可以直接在应用内删除相关记录，或通过卸载应用的方式清除全部数据。

7. **您的权利**
   - **查看**：随时在应用内查看您的所有记录。
   - **删除**：在设置页面删除单条记录或重置所有数据。
   - **彻底清除**：卸载应用将永久删除本地存储的全部数据。

8. **政策更新**
   - 当功能发生变化或法律法规有新要求时，我们可能更新本政策。更新内容将在应用内或本页面进行说明。更新后继续使用即视为您接受修订后的政策。
