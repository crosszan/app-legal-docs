# GrowMate 隐私协议

[**查看英文版本**](growmate-en.html)

1. **简介**
   GrowMate（以下简称“本应用”）由 **CrossZan** 运营，致力于保护家长及其家庭数据安全。本协议说明我们如何在不收集任何远程数据的前提下处理您在本地保存的信息，以及您可随时行使的权利。

2. **我们收集的数据**
   - 本应用不向我们或任何第三方回传、上传或同步任何数据。
   - 您在应用内输入的所有内容（如婴儿档案、成长记录等）仅保存在您的设备本地，完全由您控制。

3. **数据使用方式**
   - 所有功能均依赖您在本地记录的数据，数据不会离开您的设备。
   - 个性化设置和日志仅用于在设备上呈现功能效果，我们不会将其用于广告、画像或其他用途。

4. **数据存储与安全**
   - **本地保存**：应用产生的数据仅存储在您的设备上，我们无法远程访问或查看。
   - **iCloud 同步（可选）**：若您在系统中开启 iCloud，同步将由 Apple 负责，数据在您的 Apple ID 中加密存储，我们不持有副本。
   - **访问控制**：建议开启设备密码、Face ID 或 Touch ID，防止未授权访问。

5. **第三方共享**
   - 本应用不与任何第三方共享您的数据。
   - 如果您选择使用 iCloud 家庭共享等系统能力，该数据交换完全由 Apple 系统处理，我们不参与也无法访问。

6. **儿童隐私**
   - 本应用供家长或监护人管理婴幼儿数据，我们不会主动向儿童索取信息。
   - 如您发现儿童在未获许可的情况下提交了数据，请联系我们协助删除。

7. **您的权利**
   - **访问与导出**：可在应用内查看全部记录，并使用系统分享工具导出。
   - **更正与删除**：随时编辑或删除婴儿档案、日志与提醒；删除档案即同步删除其数据。
   - **权限管理**：可在系统设置中随时撤销通知、相册等权限。
   - **退出与数据清理**：卸载应用或在系统中清除 iCloud 数据即可停止我们对数据的访问。

8. **隐私政策更新**
   - 我们可能因功能迭代或法规要求更新本政策，并会在应用或官网醒目位置提示；更新后的继续使用被视为接受变更。

9. **联系我们**
